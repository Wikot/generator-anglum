# generator-anglum

> A [Yeoman](http://yeoman.io) generator

## Getting Started

### What is Anglum?

AngularJS + Lumen Yeoman Generator.

### Features

* AngularJS
* Lumen (Microframework from Laravel guys)
* Gulp
* Composer
* Bower

## Install

```bash
npm install -g generator-anglum
```

## Usage:

```bash
mkdir your-project-dir
cd your-project-dir
yo anglum
```
